# OpenPulse Roadmap

This document outlines the planned features and improvements for OpenPulse Analytics.

## MVP (Current)

✅ **Core Tracking**
- [x] Pageview tracking
- [x] Session tracking
- [x] Visitor counting
- [x] Basic device detection

✅ **Dashboard**
- [x] Project management (unlimited projects)
- [x] Overview metrics
- [x] Top pages
- [x] Device breakdown
- [x] Geographic data
- [x] Referrer tracking

✅ **Infrastructure**
- [x] Monorepo setup (Turborepo)
- [x] Database schema (Prisma)
- [x] tRPC API
- [x] Authentication (Better Auth)
- [x] Tracking script
